# INC-2024-002: Database Connection Pool Exhaustion

**Date**: 2024-04-02
**Service**: order-service
**Severity**: High

## Summary
Order service experienced intermittent failures during checkout.

## Root Cause
Database connection pool was exhausted because connections were not being closed properly.

## Resolution
Restarted the order-service pods and patched the connection leak.

## Prevention
Added connection leak detection, database pool dashboards, and automated alerts for connection saturation.
